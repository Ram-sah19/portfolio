## ✨ Features

- **3D Hero Scene** — Three.js torus knot + icosahedron + particle field with mouse parallax
- **3D About Card** — Interactive tilt card with animated sphere & orbiting ring
- **Tech Orbit** — Solar system-style skills visualization with hover sidebars
- **Themed Project Scenes** — Each project card has a unique Three.js 3D scene
- **Wave Contact Background** — Animated wave grid in the contact section
- **Smooth Animations** — Delta-time based rendering, lerped speed transitions
- **Responsive Design** — Mobile-first, works on all screen sizes
- **Dark Theme** — Deep dark background with gold accent color system
